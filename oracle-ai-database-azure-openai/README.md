# Oracle AI Database with Azure OpenAI

This directory contains examples and implementations for integrating Oracle Database with Azure OpenAI for AI and machine learning applications.

## Overview

This project demonstrates how to leverage Oracle Database AI capabilities with Azure OpenAI services to build intelligent applications.

## Features

- Integration with Azure OpenAI models
- Oracle Database AI Vector Search
- RAG (Retrieval Augmented Generation) implementations
- AI-powered database operations

## Prerequisites

- Oracle Database 26ai or later
- Azure Account with OpenAI service access
- Python 3.8+
