## This repository contains references and source for workshops, demos, and activations

All are open source and can be implemented with Oracle Database Container or Cloud versions, both of which are free and can be obtained [here](https://bit.ly/langflow_oracledatabase).

You can also contact Paul Parkinson with any questions [here](https://www.linkedin.com/in/pauldparkinson/).
